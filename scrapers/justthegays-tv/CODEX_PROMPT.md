# Codex Prompt

Use the files in this folder to maintain or improve the JustTheGays.tv scraper.

Start with:

- `SCRAPER_SPEC.json`
- `PERPLEXITY_TO_CODEX_HANDOFF.md`
- `JustTheGays.yml`
- `JustTheGays.py`

Requirements:

- Preserve current supported hooks unless there is a strong reason to change them
- Keep the scraper scoped to `https://justthegays.tv/`
- Prefer minimal, targeted fixes
- Update `README.md` if user-facing behavior changes
- Validate against `workflow/NEW_SCRAPER_CHECKLIST.md`
